# AffiniScore (Ionic + Angular)

App movil para gestionar afinidad y acciones en pareja.

## Stack
- Ionic 8
- Angular 20 (Standalone Components)
- Supabase
- Capacitor 8

## Levantar en desarrollo
1. Instalar dependencias:
   npm.cmd install
2. Iniciar app:
   npm.cmd start
3. Abrir:
   http://localhost:4200

## Flujo basico
- Login: /login
- Register: /register
- Actions: /actions
- Home: /home
- Catalog: /catalog
- Profile: /profile

## Notas
- En PowerShell de Windows, si `npm` falla por execution policy, usar `npm.cmd`.
